# ExtraVisionApp1

## Prerequisite
1. Visual Studio Installer에서 "C++를 사용한 데스크톱 개발" 중 "Windows 앱 SDK C++ 템플릿" 설치   
2. ".NET 데스크톱 개발"도 설치하는 것을 권장

## Workflow
- App : 애플리케이션 전체 설정
  - 타이틀바 관련 설정
- MainWindow : 애플리케이션 레이아웃 설정
  - 타이틀바 및 네비게이션뷰 설정
- HomePage : 홈페이지
  - 소개 화면
- CheatPage : 주요 로직을 실행
  - 화면 캡처
  - AI 적용
  - 윈도우 제어
  - 로그 남기기
